# Voyager

## Directory structure
```
-app/
  |__controllers/
  |__models/
  |__mailer/
  |__views/
-config/
  |__routes.js
  |__config.js
  |__passport.js (auth config)
  |__imager.js (imager config)
  |__express.js (express.js configs)
  |__middlewares/ (custom middlewares)
-public/
  |__js
     |__app.js
     |__require.js
     |__modules
        |__bindings
        |__models
        |__viewmodels
```
